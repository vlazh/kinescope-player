Changelog

v2.121.0

⚠ BREAKING CHANGES

  • IFrame Player API

    • Для исключения возможности изменения/удаления водяного знака через атрибут src iframe, параметры водяного знака следует устанавливать после создания плеера путем вызова метода setOptions.

      Например:

      playerFactory.create(params).then(function (player) {
  player.setOptions({ ui: { watermark: { text: 'watermark' } } });
});

v2.115.0

⚠ BREAKING CHANGES

  • IFrame Player API

    • Значение по умолчанию для параметра ui.watermark.mode изменено на random. См. параметры плеера.

v2.106.0

⚠ BREAKING CHANGES

  • IFrame Player API

    • Событие плеера AutoQualityChanged удалено. Вместо него используется событие QualityChanged. См. обработчики событий плеера.

    • Метод плеера getCurrentVideoQuality устарел и будет удален в будущем. Вместо него используйте метод getVideoQuality. См. методы плеера.

    • Свойство фабрики PlayerEvent устарело и будет удалено в будущем. Вместо него используйте свойство Events. См. свойства фабрики.

v2.101.0

⚠ BREAKING CHANGES

  • IFrame Regular

    • Параметр watermark игнорируется, если функциональность водяного знака не включена в панели управления. См. параметры встраивания.

v2.48.0

⚠ BREAKING CHANGES

  • IFrame Regular

    • Параметр header удален. См. параметры встраивания.

  • IFrame Player API

    • Параметр ui.openChaptersOnLoad удален. См. параметры плеера.

v2.36.0

⚠ BREAKING CHANGES

  • IFrame Regular

    • Параметр header заменен на параметр title. Для ранее созданных встраиваний параметр header все еще будет работать какое-то время. Для новых встраиваний следует использовать параметр title. См. параметры встраивания.

  • IFrame Player API

    • Параметр ui.openChaptersOnLoad устарел и через некоторое время будет удален. См. параметры плеера.

v2.27.0

⚠ BREAKING CHANGES

  • IFrame Player API

    • Параметры создания плеера files, actions перенесены в PlaylistItemOptions параметра playlist. См. создание плеера.
