Плейлисты

Поддерживаются динамичесикие плейлисты (ДП) и статические (СП).

Чтобы создать ДП, нужно указать список ID видео через урл параметр:

<iframe
  src="https://kinescope.io/playlist?video_ids=ID1,ID2,ID3[&<обычные параметры плеера>]"
  ...
></iframe>

Для СП достаточно указать ID плейлиста:

<iframe src="https://kinescope.io/pl/ID[?<обычные параметры плеера>]" ...></iframe>

Чтобы узнать список параметров плеера, см. встраивание через <iframe>

Если вы используете IFrame API, то используйте вышеуказанные примеры урл для создания плеера.
