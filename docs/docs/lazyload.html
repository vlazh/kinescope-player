Оптимизация, отложенная загрузка и автозапуск

Для экономии ресурсов и более быстрой загрузки страницы можно отложить загрузку плеера или видео.
Для этого есть несколько способов:

  • Отключение предзагрузки видео.

    В этом случае загрузится сам плеер (как и постер), но данные видео не будут загружаться до начала воспроизведения.
  • Отложенная загрузка/создание самого плеера.
  • Автозапуск при видимости на странице.
  • Автозапуск при наведении курсора на плеер.
  • Воспроизведение пока курсор над плеером.
  • Ограничение максимального качества видео при автоматическом определении подходящего качества (ABR).
