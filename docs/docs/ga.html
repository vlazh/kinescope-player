Интеграция с Google Analytics

Поддерживается интеграция с:

  • Google Analytics 4
    • Google Tag (gtag.js)
    • Google Tag Manager (gtm.js)

Google Analytics 4

Интеграция с Google Analytics 4 через Google Tag (gtag.js) работает "из коробки". Нужно лишь настроить плеер. Если плеер встраивается через <iframe>, то необходима дополнительная настройка.

Настройка Google Tag Manager

Скачайте файл контейнера, в нем содержатся необходимые настройки.
Затем импортируйте его в менеджере тегов. При импорте можно выбрать объединение настроек или перезапись. Для сохранения предыдущих настроек выбирайте объединение. Перезапись настроек удалит предыдущие настройки.

Файл контейнера содержит преднастроенный тег:

  • GA4 – Event – Kinescope Video - для Google Analytics 4;

Не забудьте опубликовать изменения.

Настройка плеера

В настройках плеера к вашему видео, необходимо включить интеграцию с Google Analytics и указать идентификатор потока данных.

Встраивание плеера через <iframe>

На страницу с плеером подключите скрипт для интеграции с google аналитикой - https://player.kinescope.io/latest/iframe.ga.js, расположив его после подключения Google Analytics.
