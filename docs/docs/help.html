Если проблема/ошибка с плеером

В случае возникновения проблемы/ошибки необходимо выполнить следующие действия:

  • Поищите решение тут.
  • Убедиться, что проблема точно с плеером, а не с вашим кодом/приложением.
  • Кратко описать суть и необходимые минимальные действия для воспроизведения проблемы.
  • Приложить (после возникновения проблемы):
    • Системный журнал плеера - из контекстного меню плеера (правой кнопкой мыши, в мобильном режиме, меню Настройки -> Еще).
    • Сообщения с консоли браузера (F12).

Для разработчиков:

  • Сделать минимальный пример с воспроизведением проблемы на codesandbox или т.п.
