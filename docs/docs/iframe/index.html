Встраивание плеера

Чтобы добавить плеер на свой сайт вы можете воспользоваться следующими способами встраивания:

  • Простое. Необязательно быть разработчиком и знать JavaScript.
  • Для разработчиков, с помощью специального IFrame API. Нужно обладать определенными знаниями.

Простое встраивание обычно осуществляется путем копирования кода встраивания из панели управления на свою страницу html. Дополнительно можно указать некоторые параметры плееру самостоятельно, см. описание простого встраивания.

Встраивание с помощью IFrame API предполагает наличие знаний и навыков разработчика и не подходит людям незнакомым с разработкой.
