Простое встраивание с помощью IFrame API без создания плеера

Многие возможности плеера доступны только при использовании IFrame Player API, но некоторые из таких возможностей могут быть доступны и при использавании простого встраивания. Для этого необходимо специальным образом подключить скрипт IFrame Player API и добавить iframe как при простом встраивании, ниже пример.

Пример встраивания

<!doctype html>
<html>
  <head>
    <!-- Скрипт IFrame Player API. -->
    <!-- URL-параметр `auto` необходим для того, чтобы все имеющиеся на момент выполнения скрипта элементы iframe -->
    <!-- были отслежены и автоматически добавлены как плееры при наличии в них такой возможности. -->
    <script defer src="https://player.kinescope.io/latest/iframe.player.js?auto"></script>
  </head>
  <body>
    <!-- url - url видео для встраивания. К нему необходимо добавить специальный URL-параметр `enableIframeApi` -->
    <!-- для обеспечения возможности добавления этого iframe как плеера в IFrame Player API. -->
    <iframe src="{url}?enableIframeApi"></iframe>
  </body>
</html>

Все необходимые параметры iframe см. в простом встраивании.

Параметры плеера IFrame Player API доступные для простого встраивания

Параметр Описание
behavior.playback Настраивается в панели управления.

Назначение и описание параметров см. в параметрах плеера
