Call To Action

Для настройки см. параметр плеера playlist.

При активации проигрывание останавливается и показывается экран поверх плеера с призывом к действию.
При нажатии пользователем на кнопку действия срабатывает событие CallAction.

Чтобы закрыть экран CTA необходимо вызвать метод closeCTA() у плеера.
